# Implement parent lookup

## Description

Create function to look up parent work item ID from the registry given a child ID.

## Implementation

```typescript
function lookupParent(registry: Registry, childId: string): string | null {
  const item = registry.work_items[childId];
  return item?.parent || null;
}
```

## Acceptance Criteria

- [x] Load registry from work-item-registry.json
- [x] Look up work item by ID
- [x] Return parent ID or null if root
- [x] Handle missing work item gracefully
